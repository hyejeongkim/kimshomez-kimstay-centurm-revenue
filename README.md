# KIMS HOMES 수익금정산 관리시스템
## KimsStay Yangyang — Revenue Settlement Management System

> ㈜킴스아이텍 | 킴스테이양양 위탁운영 수익금 정산 웹 관리시스템

### 기능
- 월별 정산 데이터 입력 및 관리 (2024~2030)
- 9개 장표 (성과보고, OTA판매자료, 고정비, 이체자료, 변동비, 변동비2, 특수비, 이체실행자료, 정산배분)
- 44개 호실 수익형/관리형/자가사용형 유형별 정산 분배
- Excel 스타일 다운로드 (색상·테두리·숫자포맷 적용)
- PDF 인쇄 (한글 완벽 지원)
- 2025년 과거 데이터 Excel 임포트
- **centermmark(위탁회원 포털) 연동**: 회원 마스터(위탁자·유형·가입비·실계좌·연락처·투숙객)를 가져오고, 월별 정산배분을 내보내 동기화

### 사용 방법
index.html 파일을 브라우저에서 열거나, GitHub Pages URL로 접속

### centermmark 연동
- **👥 회원 가져오기**: centermmark 가 내보낸 `centermmark-members.json` 을 업로드하면 위탁자·유형(수익형/관리형/자가사용형)·가입비타입·실 정산계좌·연락처·투숙객수·해지상태가 ROOMS 에 반영됩니다.
  - ⚠️ 개인정보(실계좌·연락처·생년월일)는 **브라우저(localStorage)에만** 저장되며 이 공개 저장소에는 절대 커밋하지 않습니다.
- **📤 정산 내보내기**: 이번 달 09.정산배분(최종)을 `정산배분_YYYY_MM.json` 으로 내려받아 centermmark 의 `scripts/import_settlement.py` 로 가져갈 수 있습니다.
